# Contributing to ClaimPoint

We appreciate your interest in contributing to ClaimPoint! Here's how you can help.

## Getting Started

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/yourusername/claimpoint.git`
3. **Create a branch**: `git checkout -b feature/your-feature`
4. **Setup** the project:
   ```bash
   npm run setup
   cd server && cp .env.example .env
   # Edit .env with your credentials
   ```

## Development

```bash
# Start development servers
npm run dev

# Run tests
npm test

# Build for production
npm run build
```

## Code Guidelines

- Use clear, descriptive commit messages
- Follow existing code style
- Test your changes before submitting
- Keep pull requests focused and manageable

## Pull Request Process

1. Update README.md if needed
2. Add tests for new features
3. Ensure all tests pass
4. Submit your PR with a clear description

## Questions?

Open a GitHub issue or contact us at support@claimpoint.app
